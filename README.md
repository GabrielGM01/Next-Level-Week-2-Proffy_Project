## 🚀 Começando
Baixe e instale o git:
```
https://git-scm.com/
```
Realize o clone do projeto em sua máquina:
```
git clone https://github.com/CodamaisEJ/BootCamp-Equipe2.git
```

<br>
<hr>

Node.js

```
https://nodejs.org/en/
```
